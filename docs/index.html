주말 아빠 코스
주요 기능 기술 사양

아이와의 주말,
고민 없이 행복하게

40~50대 아빠들의 어깨를 가볍게 해줄 맞춤 코스 탐색기. 아이 연령대와 아빠의 오늘의 체력 상태만 선택하면 현실적이고 부담 없는 주말 오프라인 활동 코스를 즉시 추천합니다.

⚡

초간단 2스텝 필터링

자녀의 나이(영유아부터 초등 고학년까지)와 오늘 아빠의 에너지 상태(피곤함, 보통, 충분함) 두 가지만 고르면 최적의 맞춤 여행 계획이 나타납니다.

🎒

현실적인 준비물과 예상 비용

거창한 여행 정보가 아닙니다. 아빠 지갑 사정을 고려한 합리적인 예산대별 코스와 기저귀, 물티슈, 동전 등 아빠 관점의 꼼꼼한 실전 준비물 리스트를 챙겨드립니다.

💾

Local-First 보안 및 소장

앱 내의 모든 데이터는 내 폰(Room DB)에 안전하게 보관됩니다. 가족과 꼭 다시 가고 싶은 만족스러웠던 코스는 나만의 책갈피로 저장하고 자유롭게 메모를 덧붙이세요.

기술 아키텍처 & 특징

Modern Kotlin DSL 최신 빌드 관리를 위해 Gradle Kotlin DSL을 채택하여 지속 가능한 의존성 관리 구성.
Jetpack Compose & M3 선언형 UI 툴킷과 Material Design 3 디자인 가이드를 준수하여 부드럽고 미려한 인터페이스 구현.
Room Local Database 오프라인 환경에서도 아무 장벽 없이 데이터를 불러오고 메모를 수정할 수 있는 완전 오프라인-퍼스트 설계.
Robust Fallback System 사용자가 선택한 모든 복합 조건에 100% 일치하는 코스가 없을 경우, 나이와 체력을 기준으로 가장 근접한 차선책 코스를 유연하게 제시하는 필터 시스템 구축.
